\# AI Cybersecurity Demos



This project contains simple examples that demonstrate how AI can support cybersecurity for beginner students.



\## Contents

\- \*\*article.md\*\* — An example article explaining a beginner AI cybersecurity concept.

\- \*\*examples/hidden\_text\_checker.py\*\* — A simple Python script that uses basic AI logic to detect hidden or suspicious text patterns.


\- \*\*examples/alert\_disabled\_login.py.py\*\* — Very simple script that alerts if a disabled AD account tries to login (simulated for demo)
